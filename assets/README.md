# Assets

Place the following assets in this directory:

- `icon.png` — App icon (1024x1024)
- `splash.png` — Splash screen image (1242x2436)
- `adaptive-icon.png` — Android adaptive icon foreground (1024x1024)
- `favicon.png` — Web favicon (32x32)

## Custom Fonts

For Arabic Quranic text, add the following font files here and register them in App.tsx:

- `KFGQPC-Uthmanic-Script-HAFS.ttf` — Uthmanic Quran font (recommended)
- `Amiri-Regular.ttf` — Amiri Arabic serif font
- `Scheherazade-Regular.ttf` — Scheherazade Arabic font

Download from:
- https://fonts.google.com/specimen/Amiri
- https://fonts.google.com/specimen/Scheherazade+New
- https://www.unicode.org/udhr/d/KFGQPC_Uthmanic_Script_HAFS.ttf
